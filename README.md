## Todo application

- Anyone can create, update, read as well as mark their todo as completed

### backend

follow below steps

- `cd backend`
- `npm install`
- `create .env file in backend folder and paste below line`
- `mongodb_url="YOUR_MONGOBD_URL"`
- `node index.js`

### frontend

follow below steps

- `cd frontend`
- `npm install`
- `npm run dev`

### Tech Stack

- react for frontend
- express for backend
- node for runtime
- mongodb for database
